# Exploratory Data Analysis of Vehicles 
This project performs Exploratory Data Analysis (EDA) on a vehicles dataset using Python and Pandas to uncover patterns and insights.

## Tools Used
- Python
- Pandas
- Jupyter Notebook

## Project Objectives
- Clean and prepare the vehicles dataset
- Explore important variables such as price, mileage, and year
- Identify trends and relationships in the data
- Visualize insights using charts

## Key Insights
- Distribution of vehicle prices
- Relationship between mileage and price
- Most common vehicle types
